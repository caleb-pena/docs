---
title: "Retrieve run status"
description: "Retrieves the status of the run execution based on its ID. The status can be `completed`, `failed`, or `in_progress`."
api: "GET https://api.ai21.com/studio/v1/maestro/runs/{run_id}"
contentType: "application/json"
---

## Path parameters

<ParamField path="run_id" type="string">
  A unique identifier of the run you wish to retrieve.
</ParamField>

## Returns

The [run object](/reference/run-object) matches the specified ID.

<RequestExample>

```python Python
import requests

url = "https://api.ai21.com/studio/v1/maestro/runs/{run_id}"

response = requests.request("GET", url)

print(response.text)
```


```javascript Javascript
const options = {method: 'GET'};

fetch('https://api.ai21.com/studio/v1/maestro/runs/{run_id}', options)
  .then(response => response.json())
  .then(response => console.log(response))
  .catch(err => console.error(err));
```


```bash Curl
curl --request GET \
  --url https://api.ai21.com/studio/v1/maestro/runs/{run_id}
```

</RequestExample>
